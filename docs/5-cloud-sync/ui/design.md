# ui 设计文档

> 需求: cloud-sync | 角色: ui

## 设计系统

本需求遵循统一设计系统，详见 `docs/1-account-system/ui/design.md`。

## 页面设计

云同步需求没有独立的页面，其 UI 表现嵌入在其他页面中:

### 离线提示

当用户无网络时，在页面顶部显示横幅提示:

```
[黄色背景横幅]
  图标(WifiOff) + "当前无网络连接，请联网后使用"
```

- 背景: amber-50 / amber-900
- 文字: amber-800 / amber-200
- 图标: Lucide `WifiOff`
- 位置: 页面内容区顶部，非固定定位

### 加载状态

数据请求时的加载指示:
- 首次加载: 骨架屏 (Skeleton)
- 刷新/翻页: 局部 loading spinner

## 关键决策

MVP 阶段不做离线缓存，所有操作需联网。离线时显示提示，不允许操作。
