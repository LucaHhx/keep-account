本需求遵循统一设计系统，详见 `docs/1-account-system/ui/design.md`。

## 页面设计

### 报表页 (ReportPage)

Tab 3 页面，通过图表帮用户理解收支。

**布局结构** (可滚动):
1. **页面标题**: "报表"
2. **月份切换栏**: 左箭头 + 当前月份文字 + 右箭头
3. **月度总览卡片**: 3 列等宽 (收入/支出/结余)
4. **分类占比区**: 环形图 + 分类列表
5. **趋势图区**: 柱状图

**月度总览卡片**:
- 白色圆角卡片 (rounded-xl)
- 上方: 灰色标签 ("收入"/"支出"/"结余")
- 下方: 大号数字，颜色分别为绿/红/蓝
- 金额无数据时显示灰色 "0"

**分类占比环形图**:
- 直径 180px，内圈 110px
- 内圈中心: "总支出" 标签 + 金额数字
- 颜色序列: #2563EB, #F59E0B, #10B981, #8B5CF6, #EF4444, #EC4899, #9CA3AF
- 最多显示 7 个分类，其余归入"其他"

**分类列表**:
- 每项: 色块圆点(12px) + 分类名 + 金额 + 百分比 + 右箭头
- 点击可下钻到该分类的流水列表

**趋势柱状图**:
- 蓝色柱子，高度按金额比例
- x 轴: 日期标签 (间隔显示)
- 底部图例: "每日支出"

### 分类下钻页 (CategoryDrilldownPage)

从分类占比图或列表点击进入，复用流水列表组件。
