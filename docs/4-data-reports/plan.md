# data-reports

> 数据报表：通过可视化图表帮助用户理解收支结构和趋势

## 目标

1. 用户可以查看月度收支总览（收入、支出、结余）
2. 用户可以通过分类占比图了解支出/收入的分布结构
3. 用户可以通过趋势图了解收支的变化趋势
4. 用户可以从分类占比下钻查看该分类的具体流水

## 范围

**包含:**

- 月度总览卡片（收入、支出、结余数字）
- 月份切换（左右切换月份）
- 分类占比图（饼图/环形图，展示支出按分类的占比）
- 趋势图（折线/柱状图，展示最近30天或按月的收支趋势）
- 分类下钻（点击某分类查看该分类的流水列表）
- 空状态引导（无数据时引导用户去记账）

**不包含:**

- 年度报表
- 自定义时间范围报表
- 对比分析（同比/环比）
- 报表导出
- 预算对比

## 核心用户场景

- 场景 A: 用户月底想看这个月花了多少钱，打开报表页面，看到本月收入、支出、结余的数字
- 场景 B: 用户想知道钱花哪了，查看分类占比图，发现"餐饮"占了40%
- 场景 C: 用户想看最近消费是不是在增加，查看趋势图，观察最近30天的支出曲线
- 场景 D: 用户看到"餐饮"占比太高，点击"餐饮"分类，下钻查看具体是哪些消费

## 背景

报表是记账软件的核心价值输出。用户记账的最终目的是"了解自己的钱花哪了"。通过清晰美观的图表，帮用户快速获得收支洞察，是产品核心价值主张"懂你"的直接体现。
