# 项目概览

## 基本信息

| 字段 | 内容 |
|------|------|
| 项目名称 | 记账本 (Keep Account) |
| 描述 | 面向个人的多端云同步记账应用，主打极简录入（3秒记账）与清晰美观的报表 |
| 创建日期 | 2026-02-28 |
| 阶段 | MVP |
| PRD 文件 | 云同步多端记账软件_MVP_PRD.md |

## 产品核心价值

- **快**: 最快路径完成一笔账（金额 + 分类），3次操作即可保存
- **准**: 分类、时间、备注等信息可追溯，支持编辑修正
- **懂你**: 报表清晰美观，能回答"钱花哪了、每月剩多少、趋势如何"

## 目标用户

- **个人记账用户**: 想掌握支出结构、了解收支状况的个人用户
- **轻度家庭记账**: 简单记录家庭收支（MVP 阶段为单用户账本，不做共享）

## 关键实体

| 实体 | 说明 |
|------|------|
| 用户 (User) | 注册用户，通过账号密码登录 |
| 分类 (Category) | 收支分类（系统预设 + 用户自定义），区分收入类和支出类 |
| 交易/流水 (Transaction) | 一笔收支或转账记录，包含金额、分类、时间、备注等 |

## 信息架构

- **登录/注册**: 账号密码登录、新用户注册
- **主导航 (Tab)**:
  1. **记账（首页）**: 快速记账入口
  2. **流水**: 流水列表、时间/类型筛选、明细编辑删除
  3. **报表**: 月度总览、分类占比饼图、趋势折线图、分类下钻
  4. **我的**: 分类管理（预设+自定义）、主题切换（浅色/深色）、退出登录

## 成功指标

- 新用户 2 分钟内完成注册并成功记第一笔账
- 记账操作路径: 1次输入（金额）+ 1次点击（分类）+ 1次点击（保存）
- 多端登录同账号数据一致
- 报表能正确展示收支结余、分类占比、趋势

## MVP 范围边界

### 做 (In Scope)

- 账号密码注册和登录
- 支出/收入/转账三种记账类型
- 系统预设分类 + 用户可添加自定义分类
- 流水列表（按时间/类型筛选）、编辑、删除
- 月度总览、分类占比图、趋势图
- 报表分类下钻到流水
- 浅色 + 深色主题
- 多端云同步（联网使用）

### 不做 (Out of Scope)

- 找回密码 / 邮件功能
- 多账户管理（每用户单账户）
- 离线记账 / 本地缓存
- 币种选择（固定人民币）
- 保存后撤销
- 预算/提醒、目标存钱
- 家庭共享账本
- 导入导出
- 智能分类 / 语音记账
