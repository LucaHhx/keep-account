# ui 设计文档

> 需求: transaction-list | 角色: ui

## 设计系统

本需求遵循统一设计系统，详见 `docs/1-account-system/ui/design.md`。

## 页面设计

### 流水列表页 (TransactionListPage)

Tab 2 页面，展示用户的所有收支记录。

**布局结构**:
1. **页面标题**: "流水"
2. **筛选栏**: 类型 Tab (全部/支出/收入/转账) + 日期范围选择按钮
3. **按日分组列表**: 每组有日期标题 + 收支小计 + 流水项列表

**流水项设计**:
- 左侧: 36px 圆形图标（支出红底，收入绿底）
- 中间: 分类名(主文字) + 备注(灰色小字)
- 右侧: 金额(带正负号和颜色) + 时间(灰色小字)

**金额颜色规则**:
- 支出: 红色 text-red-500，前缀 "-"
- 收入: 绿色 text-green-600，前缀 "+"
- 转账: 灰色 text-gray-500，无前缀

### 流水明细 (TransactionDetailPage)

底部弹出的模态框 (Bottom Sheet)。

**布局**:
- 拖拽指示条 (10x1px 灰色圆条，居中)
- 标题栏: "流水详情" + 编辑按钮 + 删除按钮
- 金额居中大号显示
- 信息列表: 分类、时间、备注

### 删除确认对话框

居中弹出，包含警告图标 + 提示文字 + 取消/删除两个按钮。
